Project Info

4D Engine

This project visualizes 4D shapes. An explanation of what it does via dimensional analogy would be to first think of a cube (3D). Imagine slicing this cube with a ton of parallel slices, all of which would be squares (2D). This can be used to reduce the dimension and as I have a software to render 3D shapes, I can use it to render 3D slices/cross-sections of a 4D shape. I also have future plans to also make it use a camera and project a 4D shape down to 3D.
